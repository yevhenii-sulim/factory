MY SKILLS

  • Мова розмітки

    HTML (HyperText Markup Language) - це стандартна мова розмітки для створення та структурування веб-сторінок.

  • Каскадна таблиця стилів

    CSS (Cascading Style Sheets) - це мова стилів, яка використовується для оформлення веб-документів.

  • Мова програмування

    JavaScript (JS) є мовою програмування, яка використовується для створення динамічного контенту та інтерактивності на веб-сайтах

  • Препроцесор

    "Sass" - це препроцесор таблиць стилів (CSS), що надає додатковий функціонал та зручності для написання та управління стилями веб-сайтів.

  • Сучасний фреймворк

    React - це JavaScript бібліотека для створення інтерфейсів користувача, розроблена компанією Facebook.
